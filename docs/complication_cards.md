# Volcano Rush - Complication Cards

After participants are chosen for a mission, shuffle the Complication deck and draw **1 card**. Its effect applies to that mission only. When the deck runs out, reshuffle and reuse.

### How complication extras are paid

Complication resource costs are charged to **every participant individually**, on top of the mission's base per-player cost. "+1 Rope" means each participant pays 1 Rope; a 3-player mission pays 3 Rope in total. A complication that adds "any resource" is also paid per participant.

| Card | Effect (per participant) |
|---|---|
| **Mosquito Attack** | +1 Rope per participant |
| **Wet Wood** | +1 Wood per participant, only if the mission requires Wood; otherwise no effect |
| **Collapsed Path** | +1 Stone per participant |
| **Slippery Rocks** | +2 resources per participant (any combination) |
| **Blunt Blade** | If the mission succeeds, the Knife becomes damaged afterwards |
| **Cracked Vessel** | If the mission succeeds, the Vessel becomes damaged afterwards |
| **Heat and Thirst** | +1 resource per participant (any type) |
| **Night Anxiety** | Mission requires 1 additional helper who may contribute resources but receives no points; if no willing helper is available, the mission fails |
| **Camp Panic** | Each participant may contribute at most 1 card of any single resource type to the mission |
| **Calm Breeze** | No effect |
| **Clear Sky** | No effect |

> **Sailor ability:** Draw 2 Complication cards and choose which one applies.
