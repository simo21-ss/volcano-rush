# Volcano Rush Rulebook

This folder holds the consolidated, player-facing rulebook for Volcano Rush. It is written for someone who has just opened the box and wants to learn the game from a single document. The detailed reference files in the parent `docs/` folder (`game_rules.md`, `characters.md`, `missions.md`, `complication_cards.md`, `volcano_cards.md`) remain available for designers and developers, but a player only needs to read [rules.md](rules.md).
